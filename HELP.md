## Correr App (Spring boot)

> **Importante:**
> Modificar `application-<env>.yaml` en `src/main/resources`.
  
Ir al directorio
```shell
cd webportal
```

Ejecutar la build
```shell
./gradle clean build
```

Correr el Docker
```shell
./docker-compose up --build
```
